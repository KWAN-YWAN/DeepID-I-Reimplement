# DeepID-I-Reimplement
Re-implement DeepID on Caffe framework.

Training Set: CASIA WebFace `cropped to 144x144` Gray (Washed)<br>
Validation Set: LFW `cropped to 128x128` Gray (6K Pairs)<br>
Accuracy @LFW:  `91.9667%`<br>
